
Это архитектура экранов и описание основного функционала для MVP-версии приложения. Документ определяет навигацию, ключевые сущности и пользовательские сценарии.

| ID Экрана | Название экрана | Путь доступа (Как попасть) | Краткое описание и назначение | Ключевые функции (MVP) |
| :--- | :--- | :--- | :--- | :--- |
| **Раздел 1: ЛЕНТА (FEED TAB)** |
| 1.1 | **Лента (Feed)** | Главный экран по умолчанию / Нажатие на 1-ю иконку в навбаре. | Отображает бесконечный поток всех постов от пользователей. | • Просмотр постов в виде карточек<br>• Переход к полному тексту статьи (на экран **1.3**)<br>• Переход к профилю автора (на экран **1.4**)<br>• Доступ к поиску (на экран **1.2**) |
| 1.2 | **Поиск (Search)** | Нажатие на иконку "лупы" на экране **1.1**. | Позволяет пользователям находить контент по всему приложению. | • Поле для ввода текста<br>• Отображение результатов поиска в виде списка постов<br>• Поиск по ключевым словам в заголовках и текстах |
| 1.3 | **Просмотр Статьи (Post View)** | Нажатие на карточку поста с любого экрана (напр., **1.1, 1.2, 3.1**). | Отображает полный текст выбранной статьи и позволяет взаимодействовать с ней. | • Чтение полного текста статьи<br>• Кнопка "Добавить в избранное"<br>• Просмотр списка комментариев<br>• Поле для добавления своего комментария |
| 1.4 | **Профиль другого пользователя** | Нажатие на аватар/имя автора на экране **1.1** или **1.3**. | Показывает информацию о пользователе и все его посты. | • Просмотр аватара, имени, био<br>• Список всех постов пользователя<br>• Кнопка "Подписаться / Отписаться" |
| **Раздел 2: СОЗДАНИЕ КОНТЕНТА (CREATE TAB)** |
| 2.1 | **Создание / Редактирование поста** | Нажатие на центральную кнопку (+) в навбаре. ИЛИ "Редактировать" на своем посте в **3.1**. | Форма для добавления нового поста или изменения существующего. | • Поле для ввода заголовка<br>• Текстовый редактор для основного контента<br>• Функция "Добавить обложку" (загрузка 1 фото)<br>• Кнопка "Опубликовать" / "Сохранить изменения" |
| **Раздел 3: ПРОФИЛЬ (PROFILE TAB)** |
| 3.1 | **Мой Профиль** | Нажатие на 3-ю иконку в навбаре. | "Домашний" экран пользователя со всей его информацией и контентом. | • Отображение своей информации (аватар, имя, био)<br>• Список своих постов<br>• Возможность редактировать (на **2.1**) и удалять свои посты<br>• Доступ к Избранному (на экран **3.2**)<br>• Доступ к Настройкам (на экран **3.3**) |
| 3.2 | **Избранное (Favorites)** | Кнопка/вкладка на экране **3.1**. | Список всех статей, которые пользователь сохранил для себя. | • Просмотр списка сохраненных постов<br>• Переход к статье (на экран **1.3**)<br>• Возможность убрать статью из избранного |
| 3.3 | **Настройки (Settings)** | Нажатие на иконку "шестеренки" на экране **3.1**. | Управление параметрами аккаунта и приложения. | • Пункт "Редактировать профиль" (переход на **3.4**)<br>• Кнопка "Выйти из аккаунта" (с подтверждением) |
| 3.4 | **Редактирование профиля** | Нажатие на пункт в меню на экране **3.3**. | Форма для изменения личных данных пользователя. | • Функция смены аватара<br>• Редактирование имени и био<br>• Кнопка "Сохранить" |
| **Раздел 0: ВХОД И РЕГИСТРАЦИЯ (ONBOARDING)** |
| 0.1 | **Экран Входа (Login)** | Запуск приложения неавторизованным пользователем. | Форма для аутентификации существующего пользователя. | • Поля "Email", "Пароль"<br>• Кнопка "Войти" |
